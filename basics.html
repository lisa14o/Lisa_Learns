The Basics

The Display Property

block-level elements

A block-level element extends from left to right on it's own line. Some examples of block-level elements include: p; h1,h2,etc; header; footer; section; li; ul, ol; div; aside; blockquote

inline elements

Inline elements can only occupy the space bounded by the defining inline tags. Span and a are the most common inline elements.

none

Elements set to display none will not be rendered by the page and will have no effect on the layout. All descendants will inherit this property. None is different from visibility: hidden in that hidden element boxes are still rendered but their contents are invisibile so the layout is still effected by the element.

inline-block

Inline-block elements act like inline elements except you can set a height and width or you can think of it as a block element that flows with other content rather than having its own line.

flex

Acts like a block element but works within the flexbox model.
